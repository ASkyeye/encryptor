# Encryptor

Window shellcode helper utility. Reads and encodes/encrypts shellcode in various methods to avoid signature detections. Also has various output formats.

## Encoding / Encryption methods 

- AES
- Caesar
- Insertion
- XANAX
- XOR

## Output formats 

- Base64
- C
- C#
- PowerShell
- Python
- Text
- VBA
- VB Script
